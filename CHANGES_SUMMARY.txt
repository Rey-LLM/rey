5. НОВЫЕ API ENDPOINTS (3 маршрута)
═════════════════════════════════════════════════════════════════

1️⃣ GET /api/documents
   Query параметры:
   - sortBy: 'date' | 'name' | 'priority'
   - order: 'asc' | 'desc'
   - search: string
   - category: string
   
   Возвращает:
   {
     documents: [],
     folders: {},
     stats: {},
     totalFolders: number
   }

2️⃣ GET /api/documents/project/:projectId
   Возвращает документы конкретного проекта
   
3️⃣ GET /api/documents/categories/list
   Возвращает список всех категорий
